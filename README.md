# Collaborative Study Platform

A full-stack web application that enables students to collaborate in real-time through study rooms, group chat, shared notes, and resource sharing.

## Live Demo

**Frontend:** https://collaborative-study-platform.vercel.app/

**Backend:** https://collaborative-study-platform-production.up.railway.app/

## Features

* JWT-based user authentication
* Create and join study rooms
* Real-time group chat using Socket.IO
* Shared notes collaboration
* PDF and study resource sharing
* Room member management
* Leave and delete room functionality
* Pomodoro study timer

## Tech Stack

* React.js
* TypeScript
* Node.js
* Express.js
* PostgreSQL
* Socket.IO
* JWT Authentication
* Tailwind CSS
* Vercel
* Railway
